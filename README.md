# Global YouTube Statistics Project: SQL → Power BI Dashboard (Top 1000 Channels)

## Dataset
Source: Kaggle — **Global YouTube Statistics 2023**  
https://www.kaggle.com/datasets/nelgiriyewithana/global-youtube-statistics-2023

## Project Overview
In this project, I explored the **top 1000 YouTube channels worldwide** to understand how performance differs by **country** and **content category**. I used **SQL** to clean and aggregate the dataset, then built a **Power BI dashboard** to visualize key comparisons (creator distribution, views, earnings, and most-watched genres), (I'm just doing this for practice!!).

## Goals:
1. **Which country has the most YouTubers in the top 1000?**  
2. **Which country earns the most money on average?** (average earnings by country)  
3. **Which country has the most total views?**  
4. **Which video category/genre is watched the most?** (highest views by category)

## Tools Used
- **SQL**
- **Power BI**

## Dashboard Visuals (4 Core Charts)
- **Top Countries by Number of Channels** (country count in the top 1000)
- **Average Earnings by Country** (compare average estimated earnings per country)
- **Total Views by Country** (identify where views are concentrated)
- **Total Views by Category/Genre** (find the most-watched content type)

## Standout Insight
A key finding was that the **United States** has the **highest number of YouTubers** in the top 1000, but **it does not lead in total views by country**.  
Even more surprising, **Cuba ranks as the 2nd highest country for views** despite having relatively few top channels from Cuba. This suggests an opportunity: creators could potentially increase reach by producing content that connects with **Cuban culture, topics, or audiences** (depending on the channel’s niche and goals).
